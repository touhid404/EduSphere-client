

#  EduSphere

A simple and engaging platform where users can share their ideas, insights, and thoughts through interactive posts.

🔗 **Live Site:** [Click Here](https://edusphere-aedc1.web.app/)

---

## 🚀 Features

* 📝 **Create & Share Posts** — Users can post their ideas and browse others’ contributions.
* 🔐 **Authentication** — Secure login and registration powered by Firebase.
* 🧭 **Seamless Navigation** — Smooth client-side routing using React Router.
* 🎨 **Modern UI** — Clean, responsive design built with Tailwind CSS.
* 💬 **Interactive Feed** — Scrollable feed for easy discovery and interaction.

---

## 🛠️ Tech Stack

* **Frontend:** React, Tailwind CSS, React Router, React Icons, Firebase Auth
* **Backend:** Node.js, Express.js, MongoDB Atlas

## ⚙️ Installation & Setup

### Client Setup

```bash
git clone https://github.com/touhid404/EduSphere-client.git
cd EduSphere-client
npm install

# Create a `.env` file with your Firebase config variables.

npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.
